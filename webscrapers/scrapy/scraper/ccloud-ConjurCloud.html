Tutorial

This section is a step-by-step tutorial for Privilege Cloud/PAM - Self-Hosted users to help you get started with Conjur Cloud.

PAM - Self-Hosted users: This tutorial assumes that a connection between PAM - Self-Hosted and Conjur Cloud has already been established. For details, see Set up the accounts in Privilege Cloud/PAM - Self-Hosted to work with Conjur Cloud > PAM - Self-Hosted > Step 1: Check connection to PAM - Self-Hosted.

What you will learn

In this tutorial you will wear several hats. You will learn how to:

Lesson As an/a ... You will learn how to ...

1

ISPSS tenant admin

Create Conjur Cloud admin user

2

Conjur Cloud admin

Set up the Conjur Cloud CLI

3

Conjur Cloud admin

Log in to Conjur Cloud using the Conjur Cloud CLI

4

Privilege Cloud/PAM - Self-Hosted admin

Sync a Safe containing your secrets to Conjur Cloud

5

Conjur Cloud admin

Create an identity (host) in Conjur Cloud to represent your workload

6

Conjur Cloud admin

Grant your workload permissions to access secrets

7

Conjur Cloud admin

Authenticate your workload by sending a RESTful request

8

Conjur Cloud admin/standard user

Set up your workload to fetch a secret from Conjur Cloud by sending a RESTful request from the code
